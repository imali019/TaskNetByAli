# TaskNet — System Stats Overlay for Windows

TaskNet is a lightweight and modern system monitoring overlay designed for Windows. It displays real-time system and network statistics directly on your taskbar with a clean and customizable interface.

---

## 🚀 Features

* 📶 Real-time Upload & Download speed monitoring
* 🧠 CPU usage tracking
* 💾 RAM usage monitoring
* 🎮 Dedicated GPU usage tracking
* 🌡️ Dedicated GPU temperature (accurate & stabilized)
* 🎨 Fully customizable overlay (colors, fonts, size, opacity)
* 🌗 Light/Dark mode support with auto switching
* 🧩 Clean Windows 11-style UI
* ⚡ Extremely lightweight and optimized for performance
